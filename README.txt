Sistema de Cadastro de Funcionários 💼

Descrição

Este projeto é um sistema simples de Cadastro de Funcionários em C# utilizando a abordagem de CRUD (Create, Read, Update, Delete) para armazenar e gerenciar funcionários em uma lista na memória.

Funcionalidades 🔥

Cadastrar Funcionário ✅

Listar Funcionários ✅

Buscar Funcionário por Nome 🔍

Atualizar Salário 💰

Remover Funcionário ❌

Calcular Folha de Pagamento 💸

Tecnologias Utilizadas 🛠️

C#

.NET Console Application

Como Executar o Projeto ⚙️

Pré-requisitos:

.NET SDK instalado

Clonando o Repositório

git clone https://github.com/SEU-USUARIO/sistema-cadastro-funcionarios.git
cd sistema-cadastro-funcionarios

Executando o Projeto

dotnet run

Estrutura do Projeto 📄

├── Program.cs             # Código principal
├── Funcionario.cs        # Classe Funcionario
└── README.md             # Documentação
